# roostermade
RoosterMade Platform

## Recall

`recall` is RoosterMade's project memory CLI and MCP server.

It helps humans and coding agents keep persistent, project-scoped memory across sessions by storing:
- notes
- summaries
- project context docs

Core workflow:
- `recall init` to set up `.recall/`
- `recall context` to load a project snapshot
- `recall note add` to record important work

Full docs: see `recall/README.md`.
